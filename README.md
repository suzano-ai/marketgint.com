# Marketgint.com — Agence IA pour les réseaux sociaux

## Design moderne & artistique

Une landing page haute en couleur, dynamique et conversion-focused pour **Marketgint** — l'agence qui crée du contenu social media avec l'IA mais avec un vrai âme créative.

### Caractéristiques

🎨 **Design artistique**
- Gradient meshes animés
- SVG illustrations custom
- Animations fluides et organiques
- Thème vibrant cyan/magenta/violet

📱 **Responsive design**
- Mobile-first
- Adaptable à tous les appareils
- Performance optimale

✨ **Sections**
- Hero avec call-to-action
- Services détaillés
- Pricing transparent (MAD)
- Portfolio
- Contact

🚀 **Tech**
- HTML5 pur (pas de frameworks)
- CSS3 animations & gradients
- SVG inline
- Zero dépendances

## Déploiement

```bash
# Clone
git clone https://github.com/suzano-ai/marketgint.com.git
cd marketgint.com

# Servir localement
python3 -m http.server 8000

# Ou avec Node
npx http-server
```

Accès: http://localhost:8000
